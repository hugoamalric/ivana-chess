# Ivana Chess - Utils

Tools.

## create-users

Create X users.

Usage:

```
./utils/create-users.py <api base URL> <count>`
```

## generate-markdown-conf-table

Parse `application.yml` to generate on `stdout` markdown table.

Usage:

```
./utils/generate-markdown-conf-table.py <application.yml filepaths...>`
```
