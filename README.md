# Ivana Chess

Chess game.

## How to build

```bash
./gradlew assemble
```

## How to test

```bash
./gradlew check
```

## How to run API

### With Gradle

```bash
./gradlew bootRun
```

### With Docker

```bash
docker run gleroy/ivana-chess-api
```

# How to run webapp

```bash
./gradlew serve
```
